App description

The Coder's Cheat Sheet is a web application devoted to helping the user with coding efficiency. This idea was heavily inspired by Bill Lawrence consistent postings of helpful guides on linkedIn.
  1. User will be able to search shortcuts regarding databases, languages, terminals, and deployment
  2. User will be able to post shortcuts or commands.
  3. User will have profile with bookmarking ability.
  4. More to come...
